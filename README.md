# IPL EDA Project

# Objective
Analyze IPL match data and identify factors 
affecting match outcomes and win margins.

# Data Cleaning
- Filled missing values in winner using mode.
- Dropped umpire3 column due to excessive null values.
- Handled missing values in city and player_of_match.

# Visualizations Used
Countplot
Histplot
Scatterplot
Boxplot
Heatmap
Pairplot
Barplot

# Key Insights
IPL 2012 and 2013 had the highest matches played.
Eden Gardens hosted the most IPL matches.
Most matches are won by 0-25 runs.
Most chasing teams win by 2-8 wickets.
Mumbai Indians have the highest win by runs.
Kochi Tuskers and Gujarat Lions have lowest margins.
DL method is rarely applied in IPL matches.
Win by runs and win by wickets are mutually exclusive.

# Conclusion
Venue, team strength, and batting order significantly 
influenced IPL match outcomes and win margins.
